Лист бумаги в клетку размером m × n сначала склеили в цилиндр высотой n, а затем из него вырезали некоторые из клеток. Необходимо определить, на какое число кусков распадётся цилиндр.

Input
В первой строке заданы числа m и n (1 ≤ m, n ≤ 1000). Следующие m строк (по n чисел в каждой) содержат данные клеток листа, где 1 означает, что клетку вырезали, а 0 — что клетку не вырезали. Числа в строках разделены одним или несколькими пробелами.
Output
Выведите число кусков, на которое распалась оставшаяся часть листа.
